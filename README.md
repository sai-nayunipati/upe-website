## Getting Started

### Dependencies

* Node
  
### Installing
* Run `npm install` to install the dependencies.
* Run `npm start` to start the site on localhost.
  
### Executing program
* Use [Gitname's guide on deploying React apps to GitHub Pages](https://github.com/gitname/react-gh-pages).
